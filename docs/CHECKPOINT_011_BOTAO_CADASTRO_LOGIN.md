# CHECKPOINT 011 — Botão de cadastro na tela de login

## Pedido atendido
Na tela de login, foi removida a frase confusa que dizia para criar conta clicando em “Entrar” na tela inicial.

Agora a tela de login possui um botão claro:

**Criar cadastro gratuito**

## Por que isso é importante
A tela de login também será acessada por pessoas novas. Se o usuário chegar ali sem conta, ele precisa encontrar rapidamente o caminho para se cadastrar.

Isso ajuda principalmente profissionais, empresas e lojas que receberem o link pelo WhatsApp ou Instagram e ficarem em dúvida entre entrar ou criar cadastro.

## Arquivos alterados
- `src/Login.jsx`
- `src/App.jsx`

## Resultado esperado
Na tela de login, abaixo de “Esqueci minha senha”, aparece o botão **Criar cadastro gratuito**. Ao clicar nele, o usuário é levado para a tela de cadastro.
